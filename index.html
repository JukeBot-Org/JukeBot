Introduction to JukeBot

JukeBot is an audio streaming bot for Discord which was built with specific aims in mind:

  • JukeBot is self-hosted, which gives the server administrator much more control over how (and when) their instance of JukeBot runs.
  • JukeBot is open source, which both encourages community contribution and allows anybody to customise their version of JukeBot to fit what they desire most in an audio streaming bot.
  • JukeBot is lightweight, only requiring an installation of Python 3 and an FFmpeg binary in order to function.
  • JukeBot is always free, which ensures that any features that may get added to JukeBot in the future will remain free and open to the public.
  • JukeBot is platform-agnostic, in that it runs on Windows, macOS and Linux, and can be run on anything from your personal PC at home, to a server hosted abroad!

How can I invite JukeBot to my server?

Ah, it's not that easy. JukeBot has been written with a non-centralised, self-hosted ethos in mind; you will need to host your own copy of JukeBot, either on your PC or a server somewhere.

I've written the bot with ease of setup in mind, so the steps to set it up, while they must be done manually, shouldn't be too difficult. Click here for a quickstart guide on setting up your own instance of JukeBot.
